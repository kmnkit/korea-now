# Korea Now 🇰🇷

> 日本在住の韓国人や韓国好きのための、最新韓国トレンドスポット共有アプリ

## 📖 概要

**Korea Now（コリアナウ）** は、韓国の「今」流行っているスポットや飲食店をユーザー同士で共有できるWebアプリケーションです。

### 解決する課題
- 日本在住の韓国人が、日本人に韓国旅行のおすすめを聞かれた時にすぐ答えられる
- リアルタイムの韓国トレンド情報を簡単に入手できる
- ユーザー投稿による「生の声」で信頼性の高い情報を提供

## ✨ 主な機能

- 📍 **スポット投稿**: カフェ、レストラン、観光地などを簡単に投稿
- 🔍 **検索・フィルター**: カテゴリ、エリア、タグで絞り込み
- ❤️ **リアクション**: いいね、コメント、ブックマーク
- 🛡️ **安全性**: AI + ユーザー通報 + 管理者の3層モデレーションシステム
- 🗺️ **地図連携**: Googleマップで場所を確認

## 🛠 技術スタック

### Vercel統合スタック
- **フレームワーク**: Next.js 14+ (App Router)
- **スタイリング**: Tailwind CSS + shadcn/ui
- **データベース**: Vercel Postgres
- **ストレージ**: Vercel Blob
- **キャッシュ**: Vercel KV (Redis)
- **認証**: NextAuth.js v5
- **AI**: OpenAI API (Vercel AI SDK)
- **決済**: Stripe
- **ホスティング**: Vercel

> Vercel単体でフルスタック開発可能な構成

## 💰 マネタイズ戦略

### 多様な収益源
- **ディスプレイ広告**: Google AdSense（無料ユーザー向け、韓国デザイン統合）
- **フリーミアムモデル**: 無料/プレミアム/プロプラン
- **B2B**: スポンサーシップ、データ分析レポート
- **アフィリエイト**: 旅行予約サイト連携
- **API提供**: スポット情報データ販売

### 段階的な収益化
- **初期6ヶ月**: 無料提供でコミュニティ構築
- **6ヶ月以降**: プレミアムプラン + 広告導入
- **1年目標**: 月間¥500,000-550,000の収益
- **2年目標**: 月間¥3,100,000-3,200,000の収益
