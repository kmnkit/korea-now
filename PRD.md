# Product Requirements Document (PRD)
# Korea Now - 韓国トレンドスポット共有アプリ

## 📋 プロダクト概要

**プロダクト名**: Korea Now（コリアナウ）

**コンセプト**: 日本在住の韓国人や韓国に興味がある人が、最新の韓国トレンドスポット・飲食店情報を簡単に見つけて共有できるWebアプリケーション

**タグライン**: 「今の韓国」が、ここにある

---

## 🎯 背景と解決する課題

### 課題
- 日本在住の韓国人が、日本人に韓国旅行のおすすめを聞かれることが多い
- インドア派や流行に詳しくない人は、最新情報をすぐに答えられない
- 既存の観光情報サイトは古い情報や定番スポットばかり
- リアルタイムの「今」流行っている場所が分かりにくい

### 解決策
ユーザー同士がリアルタイムで韓国の最新トレンドスポットを投稿・共有できるプラットフォームを提供し、誰でも「今」の韓国のおすすめを即座に答えられるようにする。

---

## 👥 ターゲットユーザー

### プライマリユーザー
1. **日本在住の韓国人**
   - 韓国旅行のおすすめを聞かれることが多い
   - 最新トレンドを把握しておきたい
   - 母国の情報を手軽にチェックしたい

2. **韓国旅行予定者（日本人）**
   - 定番ではなく「今」流行っている場所に行きたい
   - 現地の人のリアルな声を聞きたい
   - SNSで映える場所を探している

### セカンダリユーザー
3. **韓国カルチャー好き**
   - K-POP、韓国ドラマ、韓国グルメなどに興味がある
   - 韓国の最新トレンドを常にチェックしたい

## 🎨 主要機能

### MVP (Minimum Viable Product)

#### 1. スポット投稿機能
- **カテゴリ選択**
  - カフェ
  - レストラン
  - 観光スポット
  - ショッピング
  - ナイトライフ
  - その他

- **必須情報**
  - スポット名
  - カテゴリ
  - エリア（ソウル、釜山、済州島など）
  - 簡単な説明（200文字以内）
  - 写真（最大3枚）

- **任意情報**
  - 住所
  - 営業時間
  - 予算感
  - タグ（#カップル #一人旅 #インスタ映え など）
  - Googleマップリンク

#### 2. スポット閲覧機能
- **フィルタリング**
  - カテゴリ別
  - エリア別
  - タグ別

- **ソート機能**
  - 新着順
  - 人気順（いいね数）
  - トレンド順（最近のいいね増加率）

- **詳細表示**
  - スポット情報
  - 投稿者プロフィール（簡易）
  - いいね数
  - コメント
  - 地図表示

#### 3. リアクション機能
- **いいね機能**
  - ワンクリックで気に入ったスポットを保存
  - いいね数でトレンド判定

- **コメント機能**
  - 簡単な感想や追加情報を投稿
  - 最大200文字

- **ブックマーク機能**
  - 後で見返したいスポットを保存
  - マイページから一覧表示

#### 4. ユーザー認証
- **サインアップ/ログイン**
  - メールアドレス + パスワード
  - Googleアカウント連携
  - （将来的に）LINE、Kakao Talk連携

- **プロフィール**
  - ニックネーム
  - アイコン画像（任意）
  - 自己紹介（100文字以内、任意）

#### 5. コンテンツモデレーション機能
- **投稿前AI自動チェック**
  - 画像：不適切な画像の検出（暴力、アダルトコンテンツなど）
  - テキスト：スパム、誹謗中傷、差別的表現の検出

- **ユーザー通報機能**
  - 不適切な投稿を通報
  - 通報理由の選択（スパム、不適切なコンテンツ、誤情報など）

- **管理者レビュー**
  - 通報された投稿を管理者が確認
  - 違反コンテンツの削除・ユーザーへの警告

#### 6. 検索機能
- **キーワード検索**
  - スポット名、説明文から検索
  - エリア名、タグからも検索可能

## 🛠 技術スタック

### コア技術（Vercel統合スタック）
- **フレームワーク**: Next.js 14+ (App Router)
  - React Server Components
  - Server Actions
  - API Routes
- **スタイリング**: Tailwind CSS + shadcn/ui
- **状態管理**: React Context API / Zustand
- **TypeScript**: 型安全な開発

### データ層（Vercelエコシステム）
- **データベース**: Vercel Postgres
  - PostgreSQL互換（powered by Neon）
  - サーバーレス対応
  - 自動スケーリング
- **キャッシュ**: Vercel KV
  - Redis互換
  - セッション管理、レート制限に使用
- **ストレージ**: Vercel Blob
  - 画像・メディアファイル保存
  - CDN自動配信

### 認証・セキュリティ
- **認証**: NextAuth.js v5 (Auth.js)
  - メール/パスワード
  - Google OAuth
  - 将来: LINE、Kakao Talk
- **セッション管理**: Vercel KV (Redis)

### AI/モデレーション
- **AI SDK**: Vercel AI SDK
- **画像モデレーション**: OpenAI Vision API
- **テキストモデレーション**: OpenAI Moderation API
- **検索**: Vercel Postgres Full-Text Search

### 外部API
- **地図**: Google Maps JavaScript API
- **決済**: Stripe (プレミアム機能用)
- **メール**: Resend / SendGrid (通知用)

### デプロイ・インフラ
- **ホスティング**: Vercel
  - 自動CI/CD
  - エッジファンクション
  - グローバルCDN
  - 分析・モニタリング内蔵

## 🔒 セキュリティとモデレーション

### 投稿品質管理
1. **3段階チェックシステム**
   - Tier 1: AI自動チェック（リアルタイム）
   - Tier 2: ユーザー通報
   - Tier 3: 管理者レビュー

2. **スパム対策**
   - 同一ユーザーの連続投稿制限（1日5投稿まで）
   - 同じ内容の重複投稿検出
   - 外部リンクの制限

3. **信頼性スコアリング**
   - ユーザーの投稿が通報されずにいいねを獲得 → スコア上昇
   - 通報された投稿が多い → スコア低下
   - 新規ユーザーは初期スコアから開始

### データプライバシー
- GDPR、個人情報保護法に準拠
- ユーザーデータの暗号化
- 退会時のデータ完全削除オプション

## 💰 マネタイズ戦略

### フェーズ1: 無料提供期間（ローンチ〜6ヶ月）
初期はユーザー獲得とコミュニティ構築に集中し、全機能を無料で提供

### フェーズ2: フリーミアムモデル導入（6ヶ月〜）

#### 無料プラン
- スポット閲覧: 無制限
- 投稿: 月5件まで
- いいね・コメント: 無制限
- ブックマーク: 50件まで
- 広告表示あり

#### プレミアムプラン（月額 ¥480 / 年額 ¥4,800）
- **投稿制限解除**: 月30件まで投稿可能
- **広告非表示**: 快適な閲覧体験
- **ブックマーク無制限**: お気に入りスポットを無限に保存
- **優先表示**: 投稿がフィードで優先的に表示
- **詳細分析**: 自分の投稿のインサイト（閲覧数、いいね数の推移）
- **プレミアムバッジ**: プロフィールに表示
- **早期アクセス**: 新機能を先行体験

#### プロプラン（月額 ¥2,980 / 年額 ¥29,800）
**対象**: 韓国観光業者、インフルエンサー、旅行ブロガー
- プレミアムプランの全機能
- **投稿無制限**
- **スポンサー投稿**: 「PR」タグ付きで目立つ位置に表示
- **詳細分析ダッシュボード**: 投稿のパフォーマンス詳細
- **外部リンク許可**: 自社サイトやSNSへのリンク設置可能
- **ビジネスバッジ**: 公式ビジネスアカウント表示
- **優先サポート**: 専用問い合わせ窓口

### その他の収益源

#### 1. ディスプレイ広告（Google AdSense）
- **対象**: 無料プランユーザー
- **配置戦略**:
  - モバイル: ヘッダーバナー（320x50）、インフィード広告（3スポットごと）
  - デスクトップ: サイドバー（300x250, 300x600）、インフィード（6スポットごと）
- **ユーザー体験への配慮**:
  - 韓国スタイルの角丸・シャドウで自然に溶け込むデザイン
  - 侵入的でない配置（ポップアップ、自動再生動画なし）
  - プレミアムプランで広告非表示
- **収益予測**:
  - Year 1: ¥10,000-20,000/月（RPM ¥250-500）
  - Year 2: ¥120,000-240,000/月（月間40万PV想定）
- **詳細**: [AD_STRATEGY.md](./docs/AD_STRATEGY.md) 参照

#### 2. アフィリエイト収益
- **旅行予約**: Expedia、Booking.com、Agoda連携
- **ホテル予約**: Hotels.com、楽天トラベル連携
- **航空券**: Skyscanner、Kayak連携
- **体験予約**: Klook、KKday連携
- スポット詳細ページに「予約する」ボタンを設置

#### 3. 店舗向けスポンサーシップ
- **スポンサードスポット**: 月額 ¥10,000〜¥50,000
  - 検索結果の上位表示
  - 「スポンサー」バッジ表示
  - 専用の分析ダッシュボード

#### 4. データ分析レポート（B2B）
- **トレンドレポート**: 月額 ¥50,000〜
  - 韓国観光トレンド分析
  - エリア別人気スポットランキング
  - ユーザーデモグラフィック分析
  - 対象: 旅行代理店、韓国観光公社、マーケティング企業

#### 5. API提供（将来的）
- **スポット情報API**: 従量課金制
  - 旅行アプリへのデータ提供
  - メディアへのコンテンツ提供

### 収益目標

#### Year 1（ローンチ〜12ヶ月）
- 総収益: ¥500,000-550,000/月
- 内訳:
  - **ディスプレイ広告**: ¥10,000-20,000

  - **プロ会員**: 5人 × ¥2,980 = ¥14,900

- 総収益: ¥3,100,000-3,200,000/月
- 内訳:
  - **ディスプレイ広告**: ¥120,000-240,000（月間40万PV想定）
  - **プレミアム会員**: 500人 × ¥480 = ¥240,000
  - **プロ会員**: 50人 × ¥2,980 = ¥149,000
  - **アフィリエイト**: ¥1,200,000
  - **スポンサーシップ**: ¥1,000,000

### マネタイズ成功のための戦略

1. **価値提供の最大化**
   - 無料プランでも十分に価値を感じられる体験
   - プレミアム機能は「あると便利」レベル

2. **段階的な収益化**
   - 初期はユーザー獲得優先
   - コミュニティが成熟してからマネタイズ開始

3. **B2B2C戦略**
   - 店舗やビジネスユーザーから先に収益化
   - 一般ユーザーは緩やかに誘導

4. **透明性の確保**
   - スポンサー投稿は明確に「PR」表示
   - ユーザーの信頼を損なわない運営

## 📝 備考

### 開発優先順位
1. コア機能（投稿・閲覧・リアクション）
2. ユーザー認証
3. コンテンツモデレーション
4. 検索・フィルター機能
5. その他の機能

### リスクと対策
- **リスク**: 初期ユーザー獲得の難しさ
  - **対策**: クローズドコミュニティでのテストマーケティング、SNS活用

- **リスク**: 不適切コンテンツの流入
  - **対策**: 3層モデレーションシステム、コミュニティガイドライン明文化

- **リスク**: 情報の鮮度低下
  - **対策**: トレンドアルゴリズム、定期的な情報更新促進施策

- **リスク**: マネタイズがユーザー体験を損なう
  - **対策**: 無料プランでも十分な価値提供、段階的な収益化、透明性の確保

- **リスク**: 運営コストの増大
  - **対策**: Vercelサーバーレスアーキテクチャで変動費化、スケールに応じたコスト構造

### 技術的な優位性（Vercelスタック採用理由）

1. **開発速度の向上**
   - Next.js一つでフロントエンド・バックエンドを統合開発
   - Vercelとの完璧な統合による簡単デプロイ
   - Git連携による自動CI/CD

2. **コスト効率**
   - サーバーレスで従量課金（使った分だけ支払い）
   - 初期投資不要、スタートアップに最適
   - 無料枠で開発・テスト可能

3. **パフォーマンス**
   - グローバルエッジネットワークで高速配信
   - 自動キャッシング最適化
   - React Server Componentsでバンドルサイズ削減

4. **スケーラビリティ**
   - 自動スケーリング（トラフィック増加に自動対応）
   - データベース（Vercel Postgres）も自動スケール
   - 将来的なユーザー増加にも柔軟に対応

5. **開発者体験**
   - TypeScriptによる型安全性
   - ホットリロード、高速なビルド
   - 充実したドキュメント・コミュニティ
